Initializes PLL (Phase-locked-loop)

Parameters
sample_rateSampling rate (in Hz)
lo_frequencyLocal oscillator frequency (free-running) (in Hz)
k_pProportional gain term of PI controller (default: 0.2)
k_iIntegral gain term of PI controller (default: 8)
k_ncoGain of NCO (Numerically controlled oscillator) (voltage to frequency transfer coefficient) (default: >= than half of lo_frequency, ex. for lo_frequency=1000, k_nco=500)
halfcycles_per_symbolRate of symbols in oscillator's halfcycles (2 halfcycles = 1 full wave cycle)
frequency_filter_kgain to filter frequency_filtered variable (default: 0.994)
Returns
pll_s* Pointer to a struct to store data (aka pll object)
